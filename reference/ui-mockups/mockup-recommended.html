Design split

Library mode (Views 1 + 2): card grids, hero banners, inspector off, content-led. Default for Actresses, Titles, AV browse, and all detail pages — i.e. the consumption surfaces.

Workbench mode (View 3): table + inspector + mono fonts, persistent selection, keyboard nav. Default for Tools and triage — i.e. the administration surfaces (Duplicates, Trash, Translation, Review queue, Logs, Pending Kanji).

Same chrome (rail, topbar, ⌘K, status bar) frames both. The i toggle and view-mode switch in the topbar let any view override its default if needed.
